# Homebrew Tap

This repository contains Homebrew formulas for installing BigO and related tools.

## Installation

```bash
brew tap yourusername/tap
brew install bigo
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| bigo | Unified Claude + Ollama Agent Orchestrator |

## Updating

```bash
brew update
brew upgrade bigo
```

## Troubleshooting

If you encounter issues:

```bash
# Reinstall
brew reinstall bigo

# Check for issues
brew doctor

# View formula info
brew info bigo
```

## Contributing

This tap is automatically updated when new releases are published to the main bigo repository.
